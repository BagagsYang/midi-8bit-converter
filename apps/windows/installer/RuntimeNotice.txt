MIDI-8bit Synthesiser Windows Installer

This installer deploys the self-contained Windows release of MIDI-8bit Synthesiser.

Important:
- End users do not need the .NET SDK to run the installed application.
- This installer does require a supported 64-bit Windows version.
- The application writes exported WAV files to a folder that you choose at runtime.

If the installer reports an unsupported Windows version or architecture, use a supported x64 Windows machine for this release.
